What is CSS?

CSS stands for cascading style sheets,Â  a language for describing reusable styles for displaying markup-language-written content. Hkon Wium Lie came up with the idea in 1994. The W3C adopted CSS as a specification in December 1996, enabling web designers to modify the structure and aesthetics of their web sites today. For instance, CSS can be used to alter the font, size, and color of a specific HTML element. A developer can modify the appearance of all the sites simultaneously by linking a single CSS file to many pagess.
The example below shows an example of using CSS to define fonts, color of hyperlinks and color of the links when you hover over it. Specifically, we are only changing html tags like <a> and <body> instead of making new class or id.

body {

 font: normal 100% "trebuchet ms", Arial, Helvetica, sans-serif;

}

a {

 color: #000000;

}

a:visited {

 color: #005177;

}

a:hover {

 color: #005177;

}


Take note! It's better to save your css in a separate css file and load it on every html page to avoid.

FUN FACT

Although CSS gives us the power to create animations and transitions, we frequently use JavaScript tools to help us with adjusting a few things and managing the animation. The slide up and slide down motion is one of these well-liked effects, and most people are unaware that it can be created with only CSS.
