What is HTML?

HTML is the acronym for "Hypertext Markup Language"that is used to make websites. The hyperlinks that may be present in an HTML page are referred to as "hypertext." The term "markup language" describes the way tags are used to define the page layout and its elements. HTML coding maintains that text and graphics are formatted correctly. Without HTML, a browser would not understand how to load images or other elements or display text as an element. The HTML code for a simple webpage with a title and paragraph of text is shown in the example below.

<!doctype html>

<html>

<head>

<title>Sample Page Title</title>

</head>

<body>

Content of the Sample Page goes here.

</body>

</html>


HTML tags are the hidden keywords found in online pages that specify how the material should be formatted and displayed by your web browser.
The opening and closing parts of most tags are required. A forward-slash (/) character appears in the closing tag, which contains the same text as the opening tag. This usually reads as the "end" or "close" character.
There are some tags that are an exception to this rule and for which there is no need for a closing tag. One example is the <img> or image tag for showing images.
Each HTML file must contain the necessary tags in order for web browsers to read and interpret it.

Opening and closing tags example: 

Opening Closing
<div> </div>
<p> </p> 
<span> </span>


The opening tag defines attributes, which let you customize a tag.
For instance: <img src="image1.jpg"> or <p align="center"> ... </p>
Some attributes just need to be specified in the tag, while others are frequently assigned values using the equals sign, such as border="0" or width="50 %".
Most attributes are optional for most tags, and they are solely used to alter how a tag is presented by the browser by default. However, other elements, like the <img> tag, require characteristics like src and alt in order for the browser to correctly display the website.

Below are some examples of HTML Tags

HTML TAG Desription
<a> Create a hyperlink to help visitors get from one page to another.
<br> Create a single-line break in a document.
<div> Divide or section off the content on a web page.
<form> Create an interactive section on a web page for visitors to submit information.
<h1 to h6> Designate headings of both different sizes and importance.
<html> Tell a browser that it is viewing an HTML document.
<img> Designate a holding space for linked images.
<p> Designate a paragraph.
<script> Designate a client-side script on a web page.
<table> Create a table.

FUN FACT

HTML, or Hypertext Markup Language, was created by Tim Berners-Lee in 1990. He also created the World Wide Web which displays electronic documents, sometimes known as pages, that are created using HTML. Each page has numerous hyperlinks, which are connections to other pages. You are currently viewing one version of HTML on every web page.
